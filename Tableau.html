Tableau Expertise

Overview

Tableau is a powerful data visualization tool that allows for dynamic and interactive analysis of data. Through my Coursera courses and practical experience, I have gained extensive knowledge in Tableau, including creating dashboards, visualizations, and storytelling with data.

Key Skills

  • Creating interactive dashboards with filters, parameters, and calculated fields.
  • Building advanced visualizations: Tree maps, bar charts, line graphs, and scatter plots.
  • Designing data stories using Tableau Story Points.
  • Working with data blending, joins, and relationships in Tableau.
  • Customizing visualizations with color schemes and formatting for storytelling.
  • Using Tableau Public to publish and share dashboards.
